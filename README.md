# Parque dos Ipês - Landing Page
Site responsivo para divulgação do loteamento Parque dos Ipês, localizado na Serra do Cipó (MG).
https://parquedosipescipo.com.br/


Funcionalidades

      Design mobile-first com carrossel de fotos interativo (zoom em imagens)
      
      Formulário de contato com validação e popup de confirmação
      
      Menu hamburger responsivo + navegação desktop
      
      Integração WhatsApp (31) 99884-1410 e Google Maps
      
      Google Tag Manager (GTM-KDQ37NKJ) para analytics
      
      Deploy pronto para Netlify (netlify.toml configurado)

Tecnologias

    index.html	HTML5 + GTM	Estrutura + tracking 
    style-6.css	CSS3 Flexbox/Grid	Layout responsivo 
    script-5.js	Vanilla JS	Carrossel, form, modal 
    package-3.json	Node.js/Nodemailer	Envio de emails 
    netlify-2.toml	Netlify Functions	Build/deploy 

Responsividade Testada

    Mobile: Menu hamburger + stack vertical
    Tablet: Carrossel 2 slides + grid flex
    Desktop: Carrossel 3+ slides + layout amplo
